# 开源资源 / Open Source

我们开源了 KIGLAND TYPE-M 系列的两款面基底座模型，供社区免费下载使用。一个是可以直接打印的 STL，另一个是可以编辑网格的 Blender 工程文件。

We open-sourced two TYPE-M series face base models for the community — free to download and use. One is a ready-to-print STL, the other a fully editable Blender project.

---

## KIGLAND TYPE-M 06-1 布线模型 / Blender Mesh

Blender 工程文件（`.blend`），包含完整的布线网格结构。你可以自由编辑面型、调整结构、在此基础上设计自己的角色头壳，然后导出 STL 用于 3D 打印。

Full Blender project (`.blend`) with editable mesh topology. Modify the face shape, adjust the structure, design your own character headshell on top of it, then export STL for 3D printing.

## KIGLAND TYPE-M 06-2 面基底座 / Face Base

STL 文件，下载后可直接导入切片软件打印。适合涂装练习、打印测试、结构研究，以及快速上手 Kigurumi 头壳制作。

STL file — download and drop into your slicer. Great for painting practice, print testing, studying headshell structure, and getting started with kigurumi head making.
