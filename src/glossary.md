# 术语表 / Glossary

Kigurumi 圈层常用术语，帮助新手快速理解社区语言。

Common Kigurumi community terms to help newcomers get oriented.

---

## 产品装备 / Gear & Products

**头壳 / HeadShell** — KIG 佩戴在头部的面具/头盔，通常包含面部造型和假发。最核心的产品名词，社区最高频词。

**全头 / Full Head** — 完整包裹整个头部。包裹感强、造型饱满，佩戴负担较大。

**半头 / Half Head** — 仅覆盖面部和前部。更轻便透气，适合外出和长时间佩戴。

**二次元头 / Anime Head** — 偏动漫风格，尺寸较大，接近动漫角色的头身比。

**BJD 头 / BJD Head** — 接近真人比例的球形关节人偶风格，尺寸更友好。

**素壳 / Blank Shell** — 未经上色/妆面的空白头壳，买家可自行找妆娘定制妆容。

**妆面 / Faceup** — 头壳面部的上色/妆容设计，类似 BJD 娃娃的 faceup。

**皮 / Zentai（打底衣 / 全包衣 / 肉色衣）** — 覆盖头部以下所有皮肤的全身紧身连体衣。全包式用于棚拍，开眼/脸窗式可外出。

**视窗 / Visor（窥视孔）** — 头壳上的视觉开口，通常在眉毛位置。外面看是不透明的二次元眼睛，里面看是半透明纱窗感。

## 行为状态 / Actions & States

**变娃 / Becoming a Doll** — 社区最核心的行为描述词。穿上头壳和打底衣，从普通人变成娃娃的过程。

**开箱 / Unboxing** — 收到新头壳后的拆封展示，类似数码产品的开箱文化。

**穿戴 / Dressing Up** — 穿好全套装备（头壳 + 皮 + 假发 + 服装）的完整过程。

**出 / Going Out** — 戴着 Kig 出门/去漫展/参加活动。如"出kig"、"出展"。

**自设 / Original Character (OC)** — 自己设计的原创角色，不绑定任何现有动漫作品。

---

## 社群身份 / Community Roles

**Kiger** — KIG 玩家/爱好者的自称。

**娃娃 / Doll** — KIG 装扮形态下的自我/他人指代。社区成员通常说"我变成了娃娃"而不是"我穿了KIG"。

**真人 BJD / Living BJD（真人手办）** — 圈外人对 KIG 的常见误解——将穿戴 KIG 的真人误认为可动人偶。这也是最强的破冰流量来源。

**卤蛋 / Bald Egg** — 圈内人对未佩戴头壳状态的调侃称呼（光秃的头颅）。

**后勤 / Handler（小助手）** — 陪同 Kiger 出门的辅助人员，负责看路、提东西、应对路人。因为 KIG 穿戴后视野受限且一般不开口说话。

---

## 工艺交易 / Craft & Commerce

**定制 / Custom / Bespoke** — 根据角色资料单独制作，周期长、价格高，但独一无二。

**量产 / Production / Off-the-shelf** — 标准化生产，到手快、性价比高，但可能撞脸。

**撞脸 / Face Clash** — 两个玩家买了同一款量产头壳，头壳长得一样。

**妆娘 / Faceup Artist** — 专门为头壳上色的画师。类似 BJD 的 faceup artist。

**官妆 / Factory Faceup** — 头壳出厂时自带的妆容。

**改妆 / Re-faceup** — 把原有妆面卸掉重画。

**约妆 / Commission Faceup** — 委托妆娘定制妆容。

**头围 / Head Circumference** — 定制中的核心参数之一，决定头壳佩戴舒适度。

**肩宽 / Shoulder Width** — 决定头身比例视觉效果的关键参数。
